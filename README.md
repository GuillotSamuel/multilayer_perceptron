## Installation

1. Clone the repository
```bash
git clone https://github.com/GuillotSamuel/multilayer_perceptron
```

2. Install dependencies
```bash
make configurations
```

## Usage

Run the project using:
```bash
make
```

## Debug

Run the debuging pytest functions using:
```bash
make tests
```
